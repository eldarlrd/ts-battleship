>
> based on the classic board game with a PvE component.

## Install
```sh
bun i
```
## Develop
### Run Env
```sh
bun dev
```
### Prod Build
```sh
bun bundle
```
### Preview Prod Build
```sh
bun preview
```
### Format Code (ESLint + Prettier)
```sh
bun format
```
### Cut Orphans (Knip)
```sh
bun cut
```
## Test
### Run Watch
```sh
bun spec
```
### Coverage Report
```sh
bun cover
```
## Deploy
```sh
bun deployer
```
